
# Install CppUnit

## Prerequisite

* Install CppUnit library
* Install xsltproc

## Install CppUnit library

For Ubuntu

```sh
# install CppUnit
sudo apt install libcppunit-dev libcppunit-doc

# install xsltproc
sudo apt install xsltproc
```
