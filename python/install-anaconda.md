# Anaconda

- Go into (base) virtual environment

```sh
source ~/anaconda3/bin/activate
```

- To get out of the virtual environment anaconda-navigator

```sh
(base) $ conda deactivate
```

- Launch anaconda-navigator GUI

```sh
(base) $ anaconda-navigator
```

## Conda commands

```sh
conda --version

conda list

# envs
conda env list
```
